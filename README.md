## 🛍️ Store Page – React Product Filtering App

A responsive e-commerce store application built with React, featuring dynamic product filtering and sorting, real-time cart management, and quantity adjustment powered by structured state management.

## 🚀 Features

- 🔎 Search products by title or category

- 🏷️ Filter by category

- ↕️ Sort products:

- Most Popular

- Newest

- Price: Low to High

- Price: High to Low

- Best Rating

- 🎛 Apply filters manually with an action button

- 📱 Fully responsive grid layout

- 🧠 Controlled inputs for predictable state management

- 🧪 Mock product data for development

- 📄 Pagination to navigate through multiple pages of products


## 🧩 Tech Stack

- React (useState and useContext hooks)

- Tailwind CSS

- Material UI Icons

- Local mock data


## 🛠 Installation
git clone https://github.com/BuzzAlvin/buy-more.git
cd buy-more
npm install
npm run
